Welcome!

Sonnet is a modular and multi-function moderation bot for Discord built on top of discord.py and the LeXdPyK runtime environment.

Sonnet's features include but are not limited to:

  • Standard moderation: warns, mutes, kicks, bans
  • Advanced moderation: logging infractions, tunable automod blacklist, and antispam
  • Native hackban system, ban any user not in the guild using the normal ban command
  • Optimized code that makes response times near instant
  • Utility function: get profile on user, get guild information
  • Lightweight and easy to use starboard system
  • 0 downtime to patch most bugs
  • Modular system allowing for easy making of new modules

Sonnet made by Funey, Ultrabear, and Bredo. Website made by McKinley.
