Welcome!

Sonnet is a modular and multi-function moderation bot for Discord built on top of discord.py and the LeXdPyK runtime environment.

Sonnet's features include but are not limited to:

  • Kernel based system, load and unload modules at runtime without downtime or reloading the discord cache
  • Robust moderation system, infraction logging, powerful automod systems utilizing re2
  • Something for the bash users, sonnetsh and map allowing scripting of sonnets own commands
  • Complete modularity, load new command modules and event modules with ease and zero downtime
  • Written in python, allowing for fast development and expansion
  • Fast; internal caching allows for sub milisecond response times
  • Intentional simplicity, features that could clog up processing and potentially create buggy behavior are ommited

Sonnet comes packaged with 2 extension modules that showcase what is possible with its modularity

  • Starboard subsystem
  • ReactionRoles subsystem
