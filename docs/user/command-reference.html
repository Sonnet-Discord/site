Default Command Reference

The same information on this page can be found using the help command. This page assumes you're using the default command prefix, =

Command options in <inequality symbols> are required to use the command. Options in [square brackets] are optional.

Administration

Command Syntax Description Permission Level
=message-log <channel> Change message log Administrator
=infraction-log <channel> Change infraction log Administrator
=gdpr Enforce your GDPR rights, Guild Owner only Owner
=member-view-infractions <boolean value> Set whether members of the guild can view their own infraction count Administrator
=set-prefix <prefix> Set the guild prefix Administrator
=set-muterole <role> Set the mute role Administrator

Version

Command Syntax Description Permission Level
=version-info Prints version info on sonnet modules Everyone
=uptime Prints uptime Everyone
=statistics Prints stats about messages Everyone

Utilities

Command Syntax Description Permission Level
=ping Test connection to bot Everyone
=profile [user] Get a user's profile Everyone
=help [category] Print helptext Everyone
=avatar [user] Get avatar of a user Everyone
=serverinfo Get info on this guild Everyone
=poll Start a reaction based poll on the message Everyone
=coinflip Flip a coin Everyone

Jokes

Command Syntax Description Permission Level
=jban <uid> "Bans" a user Moderator

Automod

Command Syntax Description Permission Level
=wb-change <csv list> Change word blacklist Administrator
=add-regexblacklist <regex> Add an item to regex blacklist Administrator
=wiwb-change <csv list> Change the WordInWord blacklist Administrator
=remove-regexblacklist<regex> Remove an item from regex blacklist Administrator
=ftb-change <csv list> Change filetype blacklist Administrator
=list-blacklist List all blacklists Moderator
=blacklist-action <warn|mute|kick|ban> Set the action to occur when blacklist is broken Administrator
=blacklist-whitelist <role> Set a role that grants immunity from blacklisting Administrator
=antispam-set <messages> <seconds> Set how many messages in seconds to trigger antispam automute Administrator

Starboard

Command Syntax Description Permission Level
=starboard-channel <channel> Change starboard Administrator
=starboard-emoji <emoji> Set the starboard emoji Administrator
=starboard-enabled <boolean value> Toggle starboard on or off Administrator
=starboard-count <number> Set starboard reaction count threshold Administrator

Moderation

Command Syntax Description Permission Level
=warn <uid> [reason] Warn a user Moderator
=kick <uid> [reason] Kick a user Moderator
=ban <uid> [reason] Ban a user Moderator
=unban <uid> Unban a user Moderator
=mute <uid> [time[h|m|S]] [reason] Mute a user, defaults to no unmute (0s) Moderator
=unmute <uid> Unmute a user Moderator
=search-infractions <uid> Grab infractions of a user Moderator
=infraction-details <infractionID> Grab details of an infractionID Moderator
=delete-infraction <infractionID> Delete an infraction by infractionID Administrator
=grab-message <channelID> <messageID> Grab a message and show its contents Moderator
